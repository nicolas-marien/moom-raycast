# Moom

Control Moom

## Commands

- `List Layouts`
  - Lists available Moom layouts
  - Apply selected layout
  - Add or remove favorites with `Command+F`
- `Apply Favorite Layout`
  - Lists favorite layouts only
  - Apply quickly from a short list

## Requirements

- Moom must be installed and running
- Raycast needs permission to automate Moom (macOS may prompt on first run)
